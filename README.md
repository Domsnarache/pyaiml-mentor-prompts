## ğŸ§  System Architecture (Response Logic)

The AI's response structure is rigid to ensure quality. Every response follows this 6-Section Format:

### 1. CONCEPT & CONTEXT (The Hook)

* **Prime:** Starts with a real-world Tamil scenario (e.g., Tea shop, Bus stand) to explain *why* the concept exists.
* **Context:** Explicitly connects the topic to the AI/ML Industry.

### 2. FIRST-PRINCIPLES BREAKDOWN (The Logic)

* **The "Why":** Uses the **Feynman Technique** to explain "Atomic Logic" (Cause & Effect) in simple Tamil.
* **The "How":** Explains what happens inside the computer memory.
* **Visual:** Includes a **Minimal PEP 8 Code Snippet** immediately to show syntax.

### 3. MENTAL MODEL (The Flow)

* A text-based tree diagram or numbered sequence showing the process flow.

### 3.5. TOPIC COVERAGE (The 80/20 Rule)

* **Inner Circle (20%):** Teaches the critical "Daily Driver" concepts deeply with runnable code and line-by-line Tamil decoding.
* **Outer Circle (80%):** Lists advanced topics only as a roadmap (to prevent beginner overwhelm).

### 4. CODE INTUITION

* Comparison of **"Bad Code"** (Beginner mistakes) vs. **"Professional Code"** (Industry Standard).

### 5. DAILY PRACTICE

* Project-based homework tailored to an evening shift worker (3-4 hours availability).

### 6. KNOWLEDGE BASE

* FAQs, Common Mistakes, and Interview Questions (English Q -> Tamil A).
